# Team Todo App　（日本語の説明は下部に参照してください）

一個現代化的團隊協作待辦事項應用，支援任務分類、搜索、統計和通知功能。

![Team Todo App](https://img.shields.io/badge/React-18.3.1-blue) ![Tailwind CSS](https://img.shields.io/badge/Tailwind-3.4.1-38bdf8) ![Vite](https://img.shields.io/badge/Vite-5.4.21-646cff)

## ✨ 功能特點

### 核心功能
- 📝 **任務管理** - 創建、編輯、刪除和完成任務
- 👥 **團隊協作** - 使用團隊代碼共享任務（同設備多標籤頁同步）
- 📅 **期限管理** - 設定任務到期日，自動標記逾期任務
- 📄 **分頁顯示** - 每頁 10 個任務，支援翻頁

### 進階功能
- 🏷️ **任務分類** - 支援工作、個人、緊急、其他四種分類
- 🔍 **搜索功能** - 實時搜索任務名稱或日期
- 📊 **統計面板** - 顯示總任務數、完成率、今日到期、已逾期等統計
- 🔔 **通知提醒** - 智能檢測即將到期的任務，支援瀏覽器通知

### UI/UX
- 🎨 **現代設計** - 使用 Tailwind CSS 構建的簡潔深色主題
- 🎯 **向量圖標** - 集成 Lucide React 圖標庫
- 📱 **響應式設計** - 完美支援桌面、平板和手機
- ⚡ **快速響應** - 基於 localStorage 的本地存儲，即時響應

## 🚀 快速開始

### 環境要求

- Node.js 18.x 或更高版本
- npm 或 yarn

### 安裝步驟

1. **克隆項目**
   ```bash
   git clone <your-repo-url>
   cd todoApp
   ```

2. **安裝依賴**
   ```bash
   npm install
   ```

3. **啟動開發服務器**
   ```bash
   npm run dev
   ```

4. **訪問應用**
   
   打開瀏覽器訪問 `http://localhost:5173`

### 構建生產版本

```bash
npm run build
```

構建後的文件將在 `dist` 目錄中。

## 📖 使用指南

### 1. 加入團隊

首次使用時，輸入：
- **團隊代碼**：您的團隊唯一標識（例如：team-alpha）
- **您的名稱**：您的顯示名稱

> 💡 同一團隊的成員使用相同的團隊代碼即可共享任務

### 2. 創建任務

1. 點擊「新增任務」按鈕
2. 填寫任務名稱
3. 選擇期限日期
4. 選擇任務分類（工作/個人/緊急/其他）
5. 點擊「創建任務」

### 3. 管理任務

- ✅ **完成任務** - 點擊任務前的複選框
- ✏️ **編輯任務** - 點擊編輯按鈕
- 🗑️ **刪除任務** - 點擊刪除按鈕

### 4. 搜索和篩選

- **搜索** - 在搜索框輸入任務名稱或日期
- **分類篩選** - 點擊分類按鈕只顯示特定分類的任務
- **組合使用** - 搜索和篩選可以同時使用

### 5. 查看統計

統計面板實時顯示：
- 📋 總任務數
- ✅ 已完成任務數和完成率
- ⏰ 今日到期任務數
- ⚠️ 已逾期任務數

### 6. 啟用通知

1. 點擊右上角的鈴鐺圖標
2. 勾選「啟用瀏覽器通知」
3. 允許瀏覽器通知權限
4. 系統會自動檢測並提醒即將到期的任務

## 💾 數據存儲

### 存儲方式

應用使用 `localStorage` 存儲數據：
- 每個團隊的任務獨立存儲（key: `team_{teamCode}_tasks`）
- 團隊信息存儲（key: `team_info`）
- 通知設置存儲（key: `notification_settings`）

### 數據結構

```javascript
// 任務對象
{
  id: number,           // 唯一標識
  name: string,         // 任務名稱
  due_date: string,     // 期限日期 (YYYY-MM-DD)
  category: string,     // 分類 (work/personal/urgent/other)
  completed: boolean,   // 完成狀態
  created_at: string,   // 創建時間
  updated_at: string    // 更新時間
}
```

### 跨標籤頁同步

使用 `StorageEvent` API 實現同一瀏覽器多個標籤頁之間的實時同步。

## ⚠️ 限制說明

- **同設備限制** - 數據存儲在本地 localStorage，不支援跨設備同步
- **瀏覽器限制** - 不同瀏覽器的數據獨立，無法共享
- **團隊規模** - 適合小型團隊（5 人以下）使用

> 💡 如需跨設備協作，建議集成雲端後端服務（如 Firebase、Supabase）

# Team Todo App

タスク分類、検索、統計、通知機能をサポートする、モダンなチームコラボレーション用To Doアプリケーションです。

![React](https://img.shields.io/badge/React-18.3.1-blue) ![Tailwind CSS](https://img.shields.io/badge/Tailwind-3.4.1-38bdf8) ![Vite](https://img.shields.io/badge/Vite-5.4.21-646cff)

## ✨ 機能ハイライト

### コア機能
- 📝 **タスク管理** - タスクの作成、編集、削除、完了
- 👥 **チームコラボレーション** - チームコードを使用したタスク共有（同一デバイスの複数タブで同期）
- 📅 **期限管理** - タスクの期日設定、期限切れタスクの自動マーキング
- 📄 **ページネーション** - 1ページあたり10タスク表示、ページ送り機能サポート

### 拡張機能
- 🏷️ **タスク分類** - 仕事、個人、緊急、その他の4種類の分類をサポート
- 🔍 **検索機能** - タスク名または日付によるリアルタイム検索
- 📊 **統計ダッシュボード** - 総タスク数、完了率、本日期限、期限切れなどの統計情報を表示
- 🔔 **通知アラート** - 期日が迫っているタスクをスマートに検知し、ブラウザ通知をサポート

### UI/UX
- 🎨 **モダンデザイン** - Tailwind CSSで構築されたクリーンなダークテーマ
- 🎯 **ベクターアイコン** - Lucide Reactアイコンライブラリを統合
- 📱 **レスポンシブデザイン** - デスクトップ、タブレット、スマートフォンに完全対応
- ⚡ **高速応答** - localStorageベースのローカルストレージにより即時応答

## 📖 利用ガイド

### 1. チームへの参加

初回利用時に入力します：
- **チームコード**：あなたのチーム固有の識別子（例：team-alpha）
- **あなたの名前**：表示名

> 💡 同じチームのメンバーは同じチームコードを使用することでタスクを共有できます

### 2. タスクの作成

1. 「新規タスク」ボタンをクリック
2. タスク名を入力
3. 期限日を選択
4. タスク分類を選択（仕事/個人/緊急/その他）
5. 「タスク作成」をクリック

### 3. タスクの管理

- ✅ **タスクの完了** - タスク前のチェックボックスをクリック
- ✏️ **タスクの編集** - 編集ボタンをクリック
- 🗑️ **タスクの削除** - 削除ボタンをクリック

### 4. 検索とフィルタリング

- **検索** - 検索ボックスにタスク名または日付を入力
- **分類フィルタ** - 分類ボタンをクリックして特定の分類のタスクのみ表示
- **組み合わせ利用** - 検索とフィルタリングは同時に使用できます

### 5. 統計の確認

統計ダッシュボードにはリアルタイムで以下が表示されます：
- 📋 総タスク数
- ✅ 完了済みタスク数と完了率
- ⏰ 本日期限のタスク数
- ⚠️ 期限切れタスク数

### 6. 通知の有効化

1. 右上隅のベルアイコンをクリック
2. 「ブラウザ通知を有効にする」にチェックを入れる
3. ブラウザの通知権限を許可
4. システムが自動的に期日が迫っているタスクを検知し、アラートを表示します

## 💾 データストレージ

### ストレージ方法

アプリケーションは `localStorage` を使用してデータを保存します：
- 各チームのタスクは個別に保存されます（キー: `team_{teamCode}_tasks`）
- チーム情報は保存されます（キー: `team_info`）
- 通知設定は保存されます（キー: `notification_settings`）

### データ構造

```javascript
// タスクオブジェクト
{
  id: number,           // 一意の識別子
  name: string,         // タスク名
  due_date: string,     // 期限日 (YYYY-MM-DD)
  category: string,     // 分類 (work/personal/urgent/other)
  completed: boolean,   // 完了ステータス
  created_at: string,   // 作成時間
  updated_at: string    // 更新時間
}

### クロスタブ同期

`StorageEvent` API を使用して、同一ブラウザの複数のタブ間でリアルタイム同期を実現します。

## ⚠️ 制限事項

- **同一デバイス制限** - データはローカルのlocalStorageに保存されるため、クロスデバイス同期はサポートしていません
- **ブラウザ制限** - ブラウザが異なるとデータは独立しており、共有できません
- **チーム規模** - 小規模チーム（5人以下）での使用に適しています

> 💡 クロスデバイスでのコラボレーションが必要な場合は、クラウドバックエンドサービス（Firebase、Supabaseなど）との統合をお勧めします
